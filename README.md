# Job Board web app
Job Board web app that pulls developer jobs in Sweden from Arbetsformedlingen (Swedish Employment Agency)

![Screenshot](/job_board.png)

## Installation

Install dependencies
```bash
npm install
```

In the project folder run the below to start the app 
```bash
python script.py
```

## Frontend
**Framework:** React

**UI:** Material UI


## Backend
**Framework:** Node.js

**DB:** Redis

**API Scheduling:** Cron
